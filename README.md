# 🚀 Portfolio Project

A modern, responsive, and elegant portfolio website built to showcase my skills, projects, and professional background. Developed using cutting-edge technologies and UI libraries to ensure performance, accessibility, and a refined user experience.

## 🛠️ Built With

- [Vite](https://vitejs.dev/) – Lightning-fast build tool for modern web projects
- [React](https://reactjs.org/) – JavaScript library for building interactive UIs
- [TypeScript](https://www.typescriptlang.org/) – Strongly typed programming language for scalable JavaScript
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework for rapid UI development
- [shadcn/ui](https://ui.shadcn.com/) – Beautifully designed UI components built on top of Radix UI and Tailwind CSS

## ✨ Features

- ⚡ Blazing fast performance with Vite
- 📱 Responsive design across all devices
- 🎨 Custom styling with Tailwind CSS and shadcn/ui
- 🧠 Type safety and better developer experience with TypeScript
- ♿ Accessible and semantic HTML components
- 🌐 SEO-friendly architecture
- 📁 Modular, clean, and scalable project structure

## 🚧 Project Structure

```
src/
├── components/      # Reusable UI components
├── pages/           # Page-level components
├── assets/          # Static assets (images, fonts, etc.)
├── hooks/           # Custom React hooks
├── lib/             # Utility functions and services
├── styles/          # Tailwind and global CSS
└── main.tsx         # Entry point
```

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/rahul-dhir-0047/folio-code.git

# Navigate to the project directory
cd folio-code

# Install dependencies
npm install
```

## 🚀 Running the Project

```bash
# Start the development server
npm run dev
```

Visit `http://localhost:8080` to view your site in the browser.

## 🧪 Linting & Formatting

```bash
# Lint the codebase
npm run lint

# Format code with Prettier
npm run format
```

## 🗂️ Deployment

This project can be easily deployed to platforms like **Vercel**, **Netlify**, or **GitHub Pages**.

For example, to deploy with Vercel:

```bash
# If you have Vercel CLI installed
vercel
```
